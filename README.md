# 🌤️ Klimate - Weather App

Klimate is a modern weather application built using React that provides real-time weather updates, hourly forecasts, and multi-day predictions for cities around the world.

---

## 🚀 Features

- 🌍 Search weather by city
- 🌡️ Current weather details (temperature, humidity, wind, etc.)
- ⏱️ Hourly temperature forecast
- 📅 5-7 day weather forecast
- ⭐ Add and manage favorite cities
- ⚡ Fast data fetching with React Query
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **State Management:** @tanstack/react-query
- **Styling:** Tailwind CSS
- **API:** Weather API (e.g., OpenWeatherMap)
